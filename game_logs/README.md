## Log Files

A log file lists played pieces and their position for each game step.

The spaces are numbered from top to bottom and from left to right.

```
        7
    3      12
 0      8      16
    4      13
 1      9      17
    5      14
 2     10      18
    6      15
       11
```

The example from the main page would have the following log.
